keep domains on networksolutions, not continue their hosting and mailbox service

wushuang
2019-09-04
hosting
SSL, custom domain, github pages, mailgun, networksolution

our Hosting, Domain and Email services on networksolutions will be expired at the end of this year, the renew price is too much for us. so we explored infomations and made a reasonable choice.

Domain

keep Domains service on Networksolutions at $11.99/year/domain(at the first time they offer us $40/year/domain😂).

Hosting

we only have two small website online at this moment, we decided not to continue Networksolutions’ Hosting service, and transfer it to my VPS or even github pages. (here i transfored our manova.com and nubrikon.com to Github Pages)

setting DNS for website transformation

1. on Networksolutions, add an A Record and a CNAME Record.

edit A Record: For the “@ none” enter the following Github’s IP address: 192.30.252.153, clear the records for the “www” and “* (All others)”.
edit CNAME Record: Enter “www” in the Alias column. Leave TTL set to 7200 but on the same line, select the “@ none”.

2. on Github Pages, add “www.mydomain.com".

SSL on Github Pages

Github Pages can add SSL itself. just change the A Record to this 4 records:

185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153

Email

Networksolutions offered us $30+/year/mailbox, it was too much again.. we decided to move out. using Mailgun and gmail for receiving, forwarding emails looked like a good idea, it’s free.

sign up for Mailgun with credit card needed.

create domains, i created four domains for email service.

create credentials for each domain

create routing for forward emails from gmail

go to gmail and set every credentials you created for sending mail from gmail.

we use two gmails for forwarding custom domain emails, one for me and another one for jj.
