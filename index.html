topography on revit

  • 方案一：使用工具Infraworks, Navisworks, Fromit, 参考两个视频：

    1. Creating topography in Autodesk Revit from Infraworks，这是infraworks + formit + revit 按这个方法，最后到revit中大小需要scale，不过可以修改topography。

    2. Infraworks to Revit Workflow Site Context，这是infraworks + navisworks + revit 按这个方法，最后到revit中无法修改的到的地图，不过大小是对的。

      所以用结合两种方法，用2中大小合适的最有参照修改1的地形。
      注意，这两个地形是出自同一个infraworks中的地形。

  • 方案二：使用工具QGIS，参考视频：Part 1: Importing topographic GIS data into Revit using QGIS
