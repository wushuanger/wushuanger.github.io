networksolutions

post @ 2019-09-04

networksolutions主机、域名、邮箱到期，最后处理方法：

Domain

在networksolutions保留域名。

Hosting

  • netwoksolutions的空间不再继续。
  • 鉴于现在实际上需要的网站就一个，所以对空间的要求并不高，这个网站打算放到我的vps，或者直接托管的到github pages，再绑定下域名即可。
  • 放在github pages上，

  1. 参考networksolution给的教程操作，失败。

    • First, edit the A Records by selecting Edit A Records.

      For the “@ none” enter the following IP address 205.178.189.129(这个改成了192.30.252.153是github提供的ip）, clear the records for the “www” and “* (All others)”, then select Continue

    • Second, edit the Host Aliases (CNAME records) by selecting Edit CNAME Records.

      Enter “www” in the Alias column.

      Leave TTL set to 7200 but on the same line, enter the name servers provided by the blogging service in the Other Host section (for blogger.com, this name server would be ghs.google.com)（这里改成过githubusername.github.io，是不是跟空间有关？所以把空间的文件名改掉，还是不行）

    • 按照下面freedom的方法，重新改了这里的设置，最后可以了。就是把上面第二部中githubusername.github.io改成domain.com，一开始没有成功，过了一小时左右居然好了。

    • 突然想到，可以试一下第三方解析服务。顺便了解下cdn加速。

      Cloudflare是一家国外的CDN全球加速服务商，支持https，它提供了不同的套餐，即使是免费的用户，它的CDN加速服务也很全面，除了全球CDN加速服务外，还有DDoS保护，域名解析，反向代理等等，免费SSL证书。

      除了在本地存储图片，还可以将图片上传到一些免费的CDN服务中。比如Cloudinary提供的图片CDN服务，在Cloudinary中上传图片后，会生成对应的url地址，将地址直接拿来引用即可。

      利用官方支持为基于GitHub Pages的Hexo博客启用HTTPS

  2. 又在freedom上试了一下，参考这个，成功。完全按照视频操作，

    • 在freedom，加了1条A记录，1条CNAME记录。
    • 在github pages 加上域名：www.domain.com

Email

  • netwoksolutions的邮箱服务不再继续。暂定用mailgun转发gmail。
  • 看了Using Mailgun to Setup a Professional Custom Email for Free in Gmail,觉得这个办法可行啊。
  • 但因为mailgun需要绑定信用卡，所以找了其他服务看看行不行。试了sendgrid，参考Using a Custom Domain in Gmail for Free with Mailgun and Sendgrid，在google有一部卡住了，这个文章也没提到这个，anyway，sendgrid好像必须先有具体邮箱地址才可以呀，mailgun是直接以domain为单位，指定了域名，再在mailgun里create具体邮箱？确定是这样的话，还是mailgun合适咯，到年底再explore看看。。而且这篇文章还提到mailgun的免费用户（好像是说的没有绑定信用卡的用户？）在发邮箱的时候会让对方确认是否收来自mailgun的邮件？这个有点不顺畅了，这里有个问题就是，我注册mailgun的时候就是需要绑定信用卡的，没有不绑定这个选项啊，所以这里说的信息可能是过期信息，绑定信用卡应该很有可能就不会收到这个信息。到时候再确认下（比如到文章中提到的那个人的空间）。

过程

  • 一开始networksolutions renew报价（每年）：Hosting380刀，每个Domain40刀，每个mailbox，注意是每个mailbox，21刀，每个domain至少三个mailbox吧。。

  • 看了下一般域名服务商价格，networksolutions确实比较贵，其他服务商续费基本都在十几刀一年，比如name.com，续费价格是12.99,迁移费9.99。 namecheap，续费价格是12.98，迁移费用8.88。更多服务商价格比对。

  • explore过程中看到有人说networksolutions在你要迁的时候，他们会给你打折😂。所以又问了下，果然。。最后报价：Hosting210刀(includ two domains)，每个domain12刀，mailbox价格不变。。

  • 最后决定在networksolutions就保留域名。
